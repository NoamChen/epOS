## Highlights

* Multiboot 1.0 compliancy
* Bitmap Page frame allocator
* Higer half kernel
* Paging enabled
* ACPICA
* APIC/IOAPIC
* APIC timer
* Complete vsprintf implementation (no floats :))

## In Progress
* PS/2 Keyboard driver (8042)
* Scheduler
* Ring 3 task execution
